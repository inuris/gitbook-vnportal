# Hướng dẫn chèn liên kết vào module Liên kết web

## Mục đích

Chèn liên kết vào khu vực <mark style="color:yellow;">**Liên kết website**</mark>

<figure><img src="../.gitbook/assets/image (147).png" alt=""><figcaption></figcaption></figure>

Thao tác

1. Vào mục Quản trị Liên kết website

<figure><img src="../.gitbook/assets/image (148).png" alt=""><figcaption></figcaption></figure>

2. Click Thêm mới

<figure><img src="../.gitbook/assets/image (149).png" alt=""><figcaption></figcaption></figure>

3. Điều <mark style="color:yellow;">**Tiêu đề**</mark>, <mark style="color:yellow;">**Địa chỉ liên kết**</mark>
4. Click <mark style="color:yellow;">**Cập nhật**</mark> để hoàn thành

<figure><img src="../.gitbook/assets/image (150).png" alt=""><figcaption></figcaption></figure>
